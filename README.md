# Deux-Machina
Winter project Under BCS@IITK

##Instructions for submission:
1. Do not submit the assignmnets via File upload on the github, you should use only Git bash for submission (Don't even attempt, we will get to know ^_^)
2. For eaach week's assignmnet there will be a folder  ,eg.   Week-0, submit the assignmnet there.
3. Before submitting make a folder in this format -> < Name >_< Roll Number >_Week0  ,eg.    Ayush_240244_Week0 (for Week0 ofc).
4. Put all your files in your folder only not outside.
5. The commit message for the submissions should be ->  "Submission for Week0 < Name > < Rollnumber >"  ,eg.   "Submission for Week0 AyushShukla 240244"
